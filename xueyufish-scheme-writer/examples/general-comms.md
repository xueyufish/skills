# General Communication Guide

Use this guide when asked to write non-standard project communications (as opposed to design documents or FAQs).

## Pre-Writing Checklist

Before writing, complete these steps:

1. Ask the user about the **target audience**
2. Clarify the **purpose** of this communication
3. Confirm the desired **tone** (formal, informal, urgent, informational, etc.)
4. Verify any specific **format requirements**

## Principles

- Be clear and concise
- Use active voice
- Put important information first
- Include relevant links and references
- Maintain consistency with the team's overall communication style
